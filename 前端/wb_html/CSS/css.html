什么是CSS CSS的全称是Cascading style Sheets，层叠样式表 它用来控制html标签的样式，在美化网页中起到非常重要的作用 CSS的编写格式是健值形式，比如 color:red background-color:blue font-size:20px 冒号：左边的是属性名，冒号：右边的属性值 CSS有三种书写形式 1、行内样式：(内联样式)直接在标签的style属性中书写 style="color:red;"> 2、页内样式:在网页的style标签中书写 3、外部样式:在单独的CSS文件中书写，然后在网页中用link标签引用 CSS的属性 盒子模型 页面的布局：标准流从上到下，从左到右 position结合上下左右(子绝父相) 居中：水平居中垂直居中(标签的三种类型)
